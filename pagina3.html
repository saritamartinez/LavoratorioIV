Elementos de un sistema de informacion computacional

Equipo computacional: hardware nesesario para que el sistema de informacion funcione,
integrado por computadoras y equipo periferico conectado ah ellas.

Recurso humano: son las personas que interactuan con el sistema de informacion, suministran datos o utilizan el sistema para generar datos.

Fuente: son los datos e informacion que se introducen en el sistema, es desir, es desir, son las entrtadas requeridas para que el sistema informatico funcione.

Programas: Software ejecutado por la computadora, el cual procesa la informacion de entrada para jenerar los resultados esperados.

Telecomunicasiones: abarca el software y el hardware que permite la transmision en forma electronica de texto e imagenes, entre otros.

Politicas y reglas de operacion: aplicadas a los procedimientos y mecanismos para haceer que una aplicacion de computadora trabaje de acuerdo
con estas o regir la parte funcional del proceso administrativo de la empresa.
